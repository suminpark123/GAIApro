# GAIApro

환경데이터 대시보드를 활용한 전자상거래 웹

# 기술스택,구성

- npm
- React, Bootstrap (UI/UX)
- Node.js (서버)
- Axios 라이브러리 활용
- Mysql DB (관리자페이지 및 회원가입,로그인,댓글기능)
- Apexcharts (대시보드 그래프)
- json파일 (전자상거래 물품데이터)
- 결제 api,주소 api
